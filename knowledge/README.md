# 📚 Knowledge

Base de conocimiento del sistema: patrones probados, decisiones previas y aprendizajes por área.

## Áreas

- `software/` → patrones, arquitecturas y decisiones de ingeniería
- `databases/` → modelado, motores, rendimiento de datos
- `ui-ux/` → patrones de interfaz y experiencia de usuario
- `business/` → modelos de negocio, métricas, pricing
- `marketing/` → adquisición, posicionamiento, lanzamiento
- `startup/` → validación, MVP, aprendizajes de etapa temprana
- `3d-design/` → diseño 3D, herramientas y flujos
- `ai/` → integración de IA, prompts, modelos, agentes
