# 🧠 Adaptive AI Engineering System

## 📌 Descripción

Este proyecto es un sistema de ingeniería de software asistido por IA que toma decisiones de desarrollo basadas en el contexto del proyecto.

No es un generador de código simple, sino un sistema que analiza, razona y decide cómo construir software de forma eficiente, evitando sobreingeniería y aplicando buenas prácticas solo cuando aportan valor real.

---

## 🚀 Objetivo

Crear un sistema capaz de:

- Analizar el contexto de un proyecto
- Determinar la complejidad adecuada
- Seleccionar la arquitectura correcta
- Evitar sobreingeniería
- Aplicar principios de software de forma adaptativa
- Guiar decisiones como un ingeniero senior

---

## 🧠 Filosofía

- El contexto siempre es lo primero
- La simplicidad es la base
- La complejidad solo se usa si es necesaria
- Las reglas (SOLID, Clean Code, etc.) no son absolutas
- La IA debe pensar antes de actuar
